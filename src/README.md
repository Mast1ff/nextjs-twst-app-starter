## src

### `/src/pages`
リクエストURLに対応したページコンポーネント群

### `/src/_layouts`
ページに対応したレイアウト用ページ

### `/src/_components`
ヘッダーやフッターなどのコンポーネント単位のファイル

### `/src/_context`
React Contextの定義ファイル、storeとして利用

### `/src/_styles`
style用の変数定義やjsx styleの定義ファイル

### `/src/_interfaces`
interface, declare 用ファイル

### `/src/_lib`
共通関数ファイル

### `/src/_static`
フォントファイルやcssファイルなど、静的リソース
